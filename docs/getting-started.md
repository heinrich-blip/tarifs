# Getting Started with LoadPlan

LoadPlan is a comprehensive fleet management system designed for logistics operations. This guide will help you understand the system and get started quickly.

## Overview

LoadPlan provides the following core capabilities:

- **Load Scheduling** - Plan and schedule shipments with detailed route and timing information
- **Backload Management** - Optimize return trips by scheduling backloads (packaging, fertilizer returns)
- **Fleet Management** - Track and manage your vehicle fleet with telematics integration
- **Driver Management** - Manage driver profiles, documents, and assignments
- **Live Tracking** - Real-time GPS tracking with shareable tracking links
- **Diesel Orders** - Manage fuel orders for fleet trips
- **Reports & Analytics** - Generate insights on operations, routes, and performance

## Dashboard Overview

The Dashboard provides at-a-glance information about your operations:

### KPI Cards

- **Total Loads** - Total number of scheduled loads
- **Scheduled** - Loads waiting to be dispatched
- **In Transit** - Active loads currently on the road
- **Delivered** - Completed deliveries

### Weekly Chart

Visual representation of load volumes over time, broken down by status.

### Packaging Chart

Track packaging returns (bins, crates, pallets) from backload operations.

### Quick Actions

- **Add New Load** - Create a new load quickly
- **View Reports** - Navigate to detailed reports

## Basic Workflow

### Typical Daily Operations

1. **Morning**

   - Check Dashboard for an overview of scheduled loads
   - Review Live Tracking for in-transit vehicles
   - Check Expiry Alerts for any document issues

2. **Load Scheduling**

   - Create new loads from Dashboard or Load Planning page
   - Assign drivers and vehicles
   - Set up backloads for return trips

3. **Operations**

   - Monitor vehicles via Live Tracking
   - Share tracking links with customers
   - Create diesel orders as needed

4. **End of Day**
   - Update load statuses (mark deliveries as complete)
   - Review Reports for operational insights

## User Roles

LoadPlan supports different user access levels:

| Role           | Access                                     |
| -------------- | ------------------------------------------ |
| **Admin**      | Full access to all features                |
| **Dispatcher** | Load scheduling, tracking, diesel orders   |
| **Viewer**     | Read-only access to dashboards and reports |

---

## Key Terminology

| Term                 | Definition                                                    |
| -------------------- | ------------------------------------------------------------- |
| **Load**             | A scheduled shipment from origin to destination               |
| **Backload**         | Return trip cargo (packaging, fertilizer) going back to farms |
| **Third Party Load** | Loads for external clients/contractors                        |
| **Telematics**       | GPS tracking system integration (Telematics Guru)             |
| **Asset ID**         | Unique identifier for tracked vehicles in telematics system   |
| **COF**              | Certificate of Fitness (vehicle inspection certificate)       |
| **SVG**              | Vehicle registration certificate                              |
